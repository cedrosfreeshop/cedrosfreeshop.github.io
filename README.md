### CEDROS
App for the Cedros free Shop
https://cedrosfreeshop.github.io/

[en]
THIS APP IS DESIGNED FOR CEDROS FREE SHOP.
IT IS CONTAIN 4 FRAMES: REALES, PESOS AND INVOICE FOR EVERY FRAME.
IT IS WRITEN IN 4 LANGUAGES THAT CAN CHOOSE FROM SETTING: SPANICH, PORTUGUES, ENGLISH AND FRENCH

**FIRST FRAME:**
  - CHOOSE THE INTRO ICON LOGO THAT HAS FLOAT ANIMATION.
  - GIVE A RED ANIMATION WHEN THE USERNAME OR THE PASSWORD IS WRONG
  - SEND AN EMAIL IF YOU PUT 5 TIMES THE LOGIN INFO WRONG.
  - INTRO PANEL TO EXPLORE THE SETTINGS.
  - UPDATE BUTTON APPEAR WHEN THERE IS AN UPDATE.
  - OPTION TO DOWNGRADE FOR ANY OLD VERSION.
  - OPTION TO SHOW ALL THE CHANGELOG.

**REALES/PESOS FRAME:**
  - IT HAS A FRAME TO CALCULATE SMARTLY WHAT WE SELL EVERYDAY AND WHAT WE HAVE IN THE CASH.
  - EXPORT THE DAILY/MONTHLY/YEARLY REVIEW AT THE END OF EACH ONE WITHIN A FOLDER NAMED ACCORDING TO IT, AS A TEXT AND AN IMAGE.
  - CALCULATE THE REST FOR THE NEXT DAY OR CHOOSE WHAT YOU WANT.
  - GIVE A NOTIFICATION IF THEY CASH IS NOT GOOD BY S HALF CIRCLE APPEAR WHEN THE MOUSE ENTER IT IS AREA DOWNSIDE THE CASH VALUES.
  - ORGANIZE THE CASH, DETAILS FOR SELL VALUES, INITIAL VALUE WICH WE START THE DAY, BILLS, AGGREGATES AND CASH.
  - GIVE A SHORT VIDEO THAT SUMMARIZE THE DAY:
    * COMPARE ITS VALUE FOR THE LAST 3 YEARS VALUE OF THE SAME DAY.
    * COMPARE TO THE DAILY/MONTHLY/SAME DAY AVERAGE.
    * AN OPTION TO EXPORT THE SUMMARY FOR THE DAY.
  - A FRAME TO REVIEW THE MONTH(COMPARE IT WITH THE MONTHS BEFORE AND THE SAME MONTH LAST 3 YEARS AND GIVE THE PEAK FOR THE MONTH).
  - A FRAME TO REVIEW THE YEAR(COMPARE WITH LAST YEARS, GIVE THE PEAKS AND DETAIL THE MONTHS), AND THE OLD REGISTRED YEAR(2021-2022-2023).
  - GRAPHIC PANEL TO SHOW THE SELL OF ANY MONTH, AND OF THIS AND THE OLD YEARS WITH AN OPTION TO SAVE IT AS PNG,PDF OR SVG.
  - OPTION TO CHANGE THE APP RESOLUTION(BY DEFAULT IT WILL ADOPT THE SCREEN SIZE).
  - A NOTIFICATION BUTTON THAT GIVE YOU CERTAIN INFORMATION, LIKE THE HOLIDAYS, IF THERE A BILL TO PAY, ...
  - ALL PANEL OPEN WITH ANIMATION AND CLOSE WITH IT(BILLS, AGGREGATES, PIX, NOTIFCATION, MORE DATA).
  - A LABEL TO SHOW WHEN YOU LAST UPDATE THE CASH TO CONTROL THE ENTER MONEY.
  - A LOG FILE CREATED FOR EVERY SELL TO REVIEW IT WHEN ERROR OCCURS.
  - A BUTTON TO ASSEMBLE SET OF 100,1000 UP THEIR PHOTOS.
  - DOCK TO SWITCH BETWEEN FRAMES.
  - AUTO-UPDATE THE FRAME BY TAKING SCREENSHOT AND SEND A MESSAGE TO UPDATE THE CASH.
  - OPTION TO LOCK THE FRAME WITH PASSWORD WITH A 5 ATTEMPS TO OPEN IT(HIDE THE BACKGROUND AND EACH ELEMENTS WHEN YOU LOCK).
  - PANEL TO SAVE THE SEPARETED MONEY THAT OPEN WITH PASSWORD, AND TAKE A SCREENSHOT AT THE END OF THE WEEK.
  - DO EVERYTHING USING A SHORT KEY THAT YOU CAN FIND IN THE MENUBAR.
  - OPTION TO CLEAN THE APP CACHE.
  - OPTION TO SHOW THE BILLS OF THE DAY/WEEK/MONTH.
  - LIGHT/DARK AND BLUE MODE.
  - OPTION TO USE SOUND EFFECT(YOU CAN CHANGE IT FROM SETTING).
  - AUTO COMPLETE THE BILLS/AGGREGATES NAMES.
  - A MINI CALCULATOR TO USE FASTLY.

**THE INVOICE FRAME:**
  - CALCULATE THE SUM OF A NOTE AND GIVE THE CHANGE WITH 3 METHODS OR USING PIX.
  - THE FIRST METHOD USE THE CASH VALUES TO GIVE SMARTLY THE CHANGE.
  - THE SECOND AND THIRD METHOD IS PRACTICAL AND COMMUN.
  - IT IS ADD THE VALUES IMMEDIATLY TO THE MAIN FRAME.
  - IT CHANGE THE LAST UPDATE TIME IN THE MAIN FRAME.
  - PRINT AN OFFICAIL NOTE WITH ALL DETAIL(PRINT ONLY IF THE INVOICE IS NOT 0 AND THE USER IS NOT NULL).
  - AUTO COMPLETE THE MERCHANDISE NAME.

[sp]
ESTA APLICACIÓN ESTÁ DISEÑADA PARA CEDROS FREE SHOP.
CONTIENE 4 CUADROS: REALES, PESOS Y FACTURA POR CADA CUADRO.
ESTÁ ESCRITO EN 4 IDIOMAS QUE PUEDES ELEGIR DESDE LA CONFIGURACIÓN: ESPAÑOL, PORTUGUÉS, INGLÉS Y FRANCÉS

**PRIMER CUADRO:**
- ELIGE EL ÍCONO DE INTRODUCCIÓN QUE TIENE ANIMACIÓN FLOTANTE.
- DA UNA ANIMACIÓN ROJA CUANDO EL NOMBRE DE USUARIO O LA CONTRASEÑA ESTÉN INCORRECTOS
- ENVÍA UN CORREO ELECTRÓNICO SI PONES 5 VECES LA INFORMACIÓN DE INICIO DE SESIÓN INCORRECTA.
- PANEL DE INTRODUCCIÓN PARA EXPLORAR LA CONFIGURACIÓN.
- APARECE EL BOTÓN DE ACTUALIZACIÓN CUANDO HAY UNA ACTUALIZACIÓN.
- OPCIÓN PARA DESGRAVAR PARA CUALQUIER VERSIÓN ANTIGUA.
- OPCIÓN PARA MOSTRAR TODO EL REGISTRO DE CAMBIOS.

**MARCO REALES/PESOS:**
- CUENTA CON UN MARCO PARA CALCULAR INTELIGENTEMENTE LO QUE VENDEMOS DIARIAMENTALMENTE Y LO QUE TENEMOS EN EFECTIVO.
- EXPORTA LA RESEÑA DIARIA/MENSUAL/ANUAL AL ​​FINAL DE CADA UNO DENTRO DE UNA CARPETA CON EL NOMBRE QUE LE DEBE, COMO TEXTO Y UNA IMAGEN.
- CALCULA EL RESTO PARA EL DÍA SIGUIENTE O ESCOGE LO QUE QUIERAS.
- DA UNA NOTIFICACIÓN SI EL EFECTIVO NO ESTÁ BIEN APARECIENDO UN MEDIO CÍRCULO AL INGRESAR EL MOUSE EN EL ÁREA DE ABAJO DE LOS VALORES DEL EFECTIVO.
- ORGANIZA EL EFECTIVO, DETALLA LOS VALORES DE VENTA, VALOR INICIAL CON EL QUE EMPIEZAMOS EL DÍA, FACTURAS, AGREGADOS Y EFECTIVO.
- PROPORCIONA UN VÍDEO BREVE QUE RESUMA EL DÍA:
  * COMPARA SU VALOR CON EL VALOR DEL MISMO DÍA DE LOS ÚLTIMOS 3 AÑOS.
  * COMPARA CON EL PROMEDIO DIARIO/MENSUAL/DEL MISMO DÍA.
  * UNA OPCIÓN PARA EXPORTAR EL RESUMEN DEL DÍA.
- UN MARCO PARA REVISAR EL MES (COMPARALO CON LOS MESES ANTERIORES Y CON EL MISMO MES DE LOS ÚLTIMOS 3 AÑOS Y DA EL PICO DEL MES).
- UN MARCO PARA REVISAR EL AÑO (COMPARA CON LOS AÑOS ANTERIORES, DA LOS PICOS Y DETALLA LOS MESES), Y EL AÑO ANTIGUO REGISTRADO (2021-2022-2023).
- PANEL GRÁFICO PARA MOSTRAR LA VENTA DE CUALQUIER MES, Y DE ESTE Y AÑOS ANTERIORES CON OPCIÓN PARA GUARDARLA COMO PNG, PDF O SVG.
- OPCIÓN PARA CAMBIAR LA RESOLUCIÓN DE LA APLICACIÓN (POR DEFECTO ADOPTARÁ EL TAMAÑO DE LA PANTALLA).
- UN BOTÓN DE NOTIFICACIÓN QUE TE DA CIERTA INFORMACIÓN, COMO LOS DÍAS FESTIVOS, SI HAY UNA FACTURA QUE PAGAR, ...
- TODOS LOS PANELES SE ABRE CON ANIMACIÓN Y SE CIERRAN CON ELLA (FACTURAS, AGREGADOS, FOTOGRAFÍA, NOTIFICACIÓN, MÁS DATOS).
- UNA ETIQUETA PARA MOSTRAR CUÁNDO SE ACTUALIZÓ POR ÚLTIMA VEZ EL EFECTIVO PARA CONTROLAR EL INGRESO DE DINERO.
- UN ARCHIVO DE REGISTRO CREADO PARA CADA VENTA PARA REVISARLO CUANDO OCURRA UN ERROR.
- UN BOTÓN PARA ARMAR UN CONJUNTO DE 100,1000 FOTOS.
- DOCK PARA CAMBIAR ENTRE MARCOS.
- ACTUALIZACIÓN AUTOMÁTICA DEL MARCO TOMANDO UNA CAPTURA DE PANTALLA Y ENVIANDO UN MENSAJE PARA ACTUALIZAR EL EFECTIVO.
- OPCIÓN PARA BLOQUEAR EL MARCO CON CONTRASEÑA CON 5 INTENTOS PARA ABRIRLO (OCULTAR EL FONDO Y CADA ELEMENTO CUANDO LO BLOQUEAS).
- PANEL PARA GUARDAR EL DINERO SEPARADO QUE SE ABRE CON CONTRASEÑA, Y TOMAR UNA CAPTURA DE PANTALLA AL FINAL DE LA SEMANA.
- HAZ TODO USANDO UNA TECLA ACCESORIA QUE PUEDES ENCONTRAR EN LA BARRA DE MENÚ.
- OPCIÓN PARA LIMPIAR EL CACHÉ DE LA APLICACIÓN.
- OPCIÓN PARA MOSTRAR LOS FACTURAS DEL DÍA/SEMANA/MES.
- MODO CLARO/OSCURO Y AZUL.
- OPCIÓN PARA USAR EFECTO DE SONIDO (PUEDES CAMBIARLO DESDE LA CONFIGURACIÓN).
- COMPLETA AUTOMÁTICAMENTE LOS NOMBRES DE LAS FACTURAS/AGREGADOS.
- UNA MINI CALCULADORA PARA USAR RÁPIDAMENTE.

**EL MARCO DE FACTURA:**
- CALCULA LA SUMA DE UNA NOTA Y DAR EL CAMBIO CON 3 MÉTODOS O USANDO PIX.
- EL PRIMER MÉTODO UTILIZA LOS VALORES EN EFECTIVO PARA DAR EL CAMBIO DE FORMA INTELIGENTE.
- EL SEGUNDO Y TERCER MÉTODO SON PRÁCTICOS Y COMUNES.
- AGREGA LOS VALORES INMEDIATAMENTE AL MARCO PRINCIPAL.
- CAMBIA LA HORA DE ÚLTIMA ACTUALIZACIÓN EN EL MARCO PRINCIPAL.
- IMPRIME UNA NOTA OFICIAL CON TODOS LOS DETALLES (IMPRIME SOLO SI LA FACTURA NO ES 0 Y EL USUARIO NO ES NULO).
- COMPLETA AUTOMÁTICAMENTE EL NOMBRE DE LA MERCANCÍA.

[fr]
CETTE APPLICATION EST CONÇUE POUR LA BOUTIQUE CEDROS FREE.
ELLE CONTIENT 4 CADRES : REALES, PESOS ET FACTURE POUR CHAQUE CADRE.
ELLE EST ÉCRITE EN 4 LANGUES QUE VOUS POUVEZ CHOISIR ENTRE LES PARAMÈTRES : ESPAGNOL, PORTUGAIS, ANGLAIS ET FRANÇAIS

**PREMIÈRE IMAGE :**
- CHOISISSEZ LE LOGO DE L'ICÔNE D'INTRODUCTION QUI A UNE ANIMATION FLOTTANTE.
- DONNER UNE ANIMATION ROUGE LORSQUE LE NOM D'UTILISATEUR OU LE MOT DE PASSE EST ERRONÉ
- ENVOYER UN E-MAIL SI VOUS INSCRIVEZ 5 FOIS LES INFORMATIONS DE CONNEXION ERRONÉES.
- PANNEAU D'INTRODUCTION POUR EXPLORER LES PARAMÈTRES.
- LE BOUTON DE MISE À JOUR APPARAÎT LORSQU'IL Y A UNE MISE À JOUR.
- OPTION DE RÉTROGRADATION POUR TOUTE ANCIENNE VERSION.
- OPTION D'AFFICHER TOUS LES CHANGELOG.

**CADRE REALES/PESOS :**
- IL A UN CADRE POUR CALCULER INTELLIGEMMENT CE QUE NOUS VENDONS CHAQUE JOUR ET CE QUE NOUS AVONS EN ESPÈCES.
- EXPORTER LA REVUE QUOTIDIENNE/MENSUELLE/ANNUELLE À LA FIN DE CHAQUE DANS UN DOSSIER NOMMÉ EN FONCTION DE CELLE-CI, SOUS FORME DE TEXTE ET D'IMAGE.
- CALCULER LE RESTE POUR LE LENDEMAIN OU CHOISIR CE QUE VOUS VOULEZ.
- DONNER UNE NOTIFICATION SI L'ESPÈCE N'EST PAS BON PAR UN DEMI-CERCLE APPARAÎT LORSQUE LA SOURIS ENTRE, C'EST LA ZONE EN DESSOUS DES VALEURS EN ESPÈCES.
- ORGANISER L'ESPÈCE, LES DÉTAILS DES VALEURS DE VENTE, LA VALEUR INITIALE AVEC LAQUELLE NOUS COMMENCEZ LA JOURNÉE, LES FACTURES, LES AGRÉGATS ET L'ESPÈCE.
- DONNEZ UNE COURTE VIDÉO QUI RÉSUME LA JOURNÉE :
  * COMPAREZ SA VALEUR POUR LES 3 DERNIÈRES ANNÉES AVEC LA VALEUR DU MÊME JOUR.
  * COMPAREZ À LA MOYENNE QUOTIDIENNE/MENSUELLE/MÊME JOUR.
  * UNE OPTION POUR EXPORTER LE RÉSUMÉ DE LA JOURNÉE.
- UN CADRE POUR EXAMINER LE MOIS (COMPAREZ-LE AVEC LES MOIS PRÉCÉDENTS ET LE MÊME MOIS DES 3 DERNIÈRES ANNÉES ET DONNER LE PIC DU MOIS).
- UN CADRE POUR EXAMINER L'ANNÉE (COMPAREZ AVEC LES ANNÉES DERNIÈRES, DONNER LES PICS ET DÉTAILLER LES MOIS), ET L'ANCIENNE ANNÉE ENREGISTRÉE (2021-2022-2023).
- PANNEAU GRAPHIQUE POUR MONTRER LA VENTE DE N'IMPORTE QUEL MOIS, ET DE CETTE ANNÉE ET DES ANCIENNES ANNÉES AVEC UNE OPTION POUR L'ENREGISTRER AU FORMAT PNG, PDF OU SVG.
- OPTION POUR CHANGER LA RÉSOLUTION DE L'APPLICATION (PAR DÉFAUT, ELLE ADOPTERA LA TAILLE DE L'ÉCRAN).
- UN BOUTON DE NOTIFICATION QUI VOUS DONNE CERTAINES INFORMATIONS, COMME LES JOURS FÉRIÉS, S'IL Y A UNE FACTURE À PAYER, ...
- TOUS LES PANNEAUX S'OUVRENT AVEC UNE ANIMATION ET SE FERMENT AVEC ELLE (FACTURES, AGRÉGATS, PIX, NOTIFICATION, PLUS DE DONNÉES).
- UNE ÉTIQUETTE POUR MONTRER QUAND VOUS AVEZ MIS À JOUR LA DERNIÈRE FOIS LA CASH POUR CONTRÔLER LA SAISIE D'ARGENT.
- UN FICHIER JOUR CRÉÉ POUR CHAQUE VENTE POUR LE RÉVISER EN CAS D'ERREUR.
- UN BOUTON POUR ASSEMBLER UN ENSEMBLE DE 100,1000 PHOTOS.
- DOCK POUR BASCULER ENTRE LES CADRES.
- MISE À JOUR AUTOMATIQUE DU CADRE EN PRENANT UNE CAPTURE D'ÉCRAN ET ENVOYER UN MESSAGE POUR METTRE À JOUR LA CASH.
- OPTION POUR VERROUILLER LE CADRE AVEC MOT DE PASSE AVEC 5 TENTATIVES D'OUVERTURE (MASQUER L'ARRIÈRE-PLAN ET CHAQUE ÉLÉMENT LORSQUE VOUS VERROUILLEZ).
- PANNEAU POUR SAUVEGARDER L'ARGENT SÉPARÉ QUI S'OUVRE AVEC MOT DE PASSE, ET PRENDRE UNE CAPTURE D'ÉCRAN À LA FIN DE LA SEMAINE.
- FAITES TOUT EN UTILISANT UNE TOUCHE COURTE QUE VOUS POUVEZ TROUVER DANS LA BARRE DE MENU.
- OPTION POUR NETTOYER LE CACHE DE L'APPLICATION.
- OPTION POUR AFFICHER LES FACTURES DU JOUR/SEMAINE/MOIS.
- MODE CLAIR/FONCÉ ET BLEU.
- OPTION POUR UTILISER L'EFFET SONORE (VOUS POUVEZ LE MODIFIER À PARTIR DES PARAMÈTRES).
- COMPLÉTER AUTOMATIQUEMENT LES NOMS DES FACTURES/AGRÉGATS.
- UNE MINI CALCULATRICE À UTILISER RAPIDEMENT.

**CADRE DE FACTURE :**
- CALCULEZ LA SOMME D'UN BILLET ET RENDEZ LA MONNAIE AVEC 3 MÉTHODES OU EN UTILISANT PIX.
- LA PREMIÈRE MÉTHODE UTILISE LES VALEURS EN ESPÈCES POUR RENDRE INTELLIGEMMENT LA MONNAIE.
- LA DEUXIÈME ET LA TROISIÈME MÉTHODE SONT PRATIQUES ET COMMUNES.
- IL S'AGIT D'AJOUTER IMMÉDIATEMENT LES VALEURS AU CADRE PRINCIPAL.
- IL CHANGE L'HEURE DE LA DERNIÈRE MISE À JOUR DANS LE CADRE PRINCIPAL.
- IMPRIMEZ UNE NOTE OFFICIELLE AVEC TOUS LES DÉTAILS (IMPRIMEZ UNIQUEMENT SI LA FACTURE N'EST PAS 0 ET QUE L'UTILISATEUR N'EST PAS NUL).
- COMPLÉTEZ AUTOMATIQUEMENT LE NOM DE LA MARCHANDISE.
